# Helm + Argo CD GitOps Project

## Overview
This project demonstrates a complete GitOps workflow using Helm and Argo CD.

## Tech Stack
- Docker
- Kubernetes
- Helm
- Argo CD
- GitHub

## Flow
1. Build Docker image
2. Push image to Docker Hub
3. Define Kubernetes resources using Helm
4. Deploy via Argo CD using GitOps

## GitOps Benefits
- Automated deployments
- Self-healing
- Easy rollback
- Version-controlled infrastructure
